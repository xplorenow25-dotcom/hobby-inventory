# 🛡️ Hobby Vault | Premium Inventory Manager

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Status](https://img.shields.io/badge/status-Alpha-orange.svg)

**Hobby Vault** is a high-end digital sanctuary for collectors. Whether you collect sneakers, vintage games, or designer clothes, Hobby Vault allows you to track your "Portfolio Value" with a premium, glassmorphism UI.



## ✨ Key Features
* **💎 Premium UI/UX:** A modern, dark-mode interface designed for the Western market.
* **📊 Value Tracking:** Monitor the "Net Worth" of your hobbies in real-time.
* **📸 Document Vault:** Securely store photos of receipts and warranty cards.
* **🔍 Smart Filters:** Quickly find items by name, category, or purchase date.
* **📄 Insurance Exports:** Generate a PDF report of your belongings for insurance purposes.

## 🚀 Built With
* **HTML5** - Semantic structure
* **CSS3** - Custom properties and Glassmorphism effects
* **JavaScript** - Dynamic filtering and search logic
